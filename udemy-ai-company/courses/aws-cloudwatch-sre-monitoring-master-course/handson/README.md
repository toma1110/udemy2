# Handson Design

このディレクトリは、長尺旗艦コース用の統合ハンズオン設計領域です。

## Scope

- ハンズオン前安全チェック
- Logs Insights調査クエリ
- CloudWatch Custom Metrics投入
- Alarm、Composite Alarm、SNS、Dashboard
- Runbook初動対応
- SLI/SLO設計ワークシート
- Capstone障害調査シミュレーション
- 削除後のコスト確認

## Rule

- AWSリソース作成、更新、削除を伴う検証はCEO承認後にのみ実行する
- README通りに再現できることを合格条件にする
- CloudFormationは教材ハンズオン用に限定する
- 実運用IaCはCDKまたはTerraform推奨として扱う
